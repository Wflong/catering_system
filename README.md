# catering_system
本系统的定位：</br>
自主点餐，自主结账，账单邮件发送，账单详情查询</br>
</br>
关于代码结构说明：</br>
1、bin存放可执行py脚本文件</br>
2、config存放配置文件，涉及到频繁操作的，需要频繁读写的文件，建议使用json；涉及到系统常量配置，建议使用py文件。</br>
3、web存放前端代码</br>
</br>
关于代码提交规范</br>
新加脚本/文件夹请使用：====add:xxxx====</br>
修复bug请使用：====fix:xxx====</br>
修改脚本请使用：====modify：xxx====</br>
</br>
关于分支</br>
请自建本地分支，如果自测无误，合并到master主分支</br>
